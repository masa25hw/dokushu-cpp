# CMake

## VSCode settings

Install CMake.

```shell
$ sudo apt install cmake
```

Search cake path.

<!-- cSpell:disable -->
```shell
$ whereis cmake
```
<!-- cSpell:enable -->

Set cmakePath.

```yaml
"cmake.cmakePath": "/path/to/cmake"
```

Example:
```yaml
"cmake.cmakePath": "/usr/bin/cmake"
```
